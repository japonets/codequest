<h2>📌 Sobre</h2>
<p>
  O <strong>CODEQUEST</strong> é uma iniciativa voltada ao ensino de
  desenvolvimento de jogos utilizando a engine <strong>Godot</strong>,
  com foco em aprendizado prático, acessível e gratuito.
</p>

---

<h2>📂 Conteúdo</h2>
<p>Este repositório reúne materiais públicos do projeto, como:</p>

<ul>
  <li>Exemplos de projetos</li>
  <li>Códigos utilizados nas aulas</li>
  <li>Materiais de apoio</li>
  <li>Estrutura de aprendizado</li>
</ul>

---

<h2>🎯 Objetivo</h2>
<ul>
  <li>Ensinar lógica de programação através de jogos</li>
  <li>Introduzir desenvolvimento com Godot</li>
  <li>Facilitar o acesso ao aprendizado em tecnologia</li>
  <li>Estimular criatividade e resolução de problemas</li>
</ul>
